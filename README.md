SecureVerify: Decentralized Certificate Verification
SecureVerify is a Next.js-based decentralized application (dApp) designed to issue and verify digital certificates with absolute transparency and security using the Polygon blockchain.
🚀 The Problem
Traditional paper or PDF certificates are easily forged and difficult for employers or institutions to verify quickly. This leads to credential fraud and a slow manual verification process.
✨ Our Solution
By leveraging Blockchain and IPFS, SecureVerify ensures:
Immutability: Once a certificate is issued, its data cannot be changed.
Instant Verification: Anyone with a link or QR code can verify the authenticity against the blockchain.
Decentralized Storage: Certificate assets are stored on Pinata (IPFS) to ensure they are never lost or deleted.
🛠️ Tech Stack
Frontend: Next.js, Tailwind CSS, Lucide React.
Authentication: Firebase (Email/Password Login).
Blockchain: Polygon Amoy Testnet.
Storage: Pinata IPFS.
Smart Contracts: Solidity (Hardhat)
